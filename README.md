# Queue Management System

A simple Queue Management System built using PHP and MySQL.

This system allows offices or service centers to manage customer queues efficiently.

## Features
- Generate queue numbers
- Display current queue
- Track queue progress
- Notification sound when next number is called
- Simple admin interface

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Install XAMPP or any PHP server.
2. Copy the project folder to htdocs.
3. Import the database file into phpMyAdmin.
4. Open localhost in your browser.
